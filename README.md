# LeetCode Auto Sync

This repository is configured to store LeetCode submissions automatically using a browser extension such as **LeetHub v2**.

## Recommended structure

- `leetcode/easy/`
- `leetcode/medium/`
- `leetcode/hard/`

Each accepted submission can be committed automatically by the extension into its own problem folder.

## One-time setup (LeetHub v2)

1. Install LeetHub v2 extension.
2. Connect GitHub account.
3. Select repository: `devansh2991/LeetCode`.
4. Select branch: `main`.
5. Log in to LeetCode in the same browser.
6. Enable auto-sync in extension settings.

After setup, every accepted submission should be pushed here automatically.
